Quick Information (Taken from DBC 9.3 Module)

jQuery is a JavaScript library (an existing JavaScript file of functions that you can call in your projects, in other words) that helps you do the following with fewer lines of code (which are often more readable):

  • Add event listeners for clicking, hovering, and other user actions
  • Add or remove CSS classes in response to events
  • Make elements fade in or out, slide up or down, or otherwise transition from visible to invisible
  • Grab values from forms
  • ... and lots more!
